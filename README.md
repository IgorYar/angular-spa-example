# Angular SPA Example

## Task: create SPA that meets the following requirements

### List of technologies
- Angular
- Angular Material

### App content
- Registration Form (main page)
  - Dynamic validations
- Login Form
  - Dynamic validations
- Posts list
  - Id
  - Title (inline editable)
  - Body (inline editable)
  - Actions
    - Add
    - Edit
    - Delete
  - Filter by text
  - Validation
  - Pagination
  - Autoreloading (every 2 min when page is active)
